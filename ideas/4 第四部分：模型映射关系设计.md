## 第四部分：元模型映射关系设计

### 4.1 映射关系总览

四大视角之间存在六组两两映射关系，构成一个完整的知识网络：

```
                    ┌──────────────┐
                    │  业务视角     │
                    │  Business    │
                    └──────┬───────┘
                   ╱       │        ╲
                 ╱         │          ╲
        ┌──────╱───┐       │       ┌───╲──────┐
        │ B↔P 映射 │       │       │ B↔T 映射 │
        └──────╲───┘       │       └───╱──────┘
                 ╲         │         ╱
                   ╲       │       ╱
    ┌──────────────┐╲      │     ╱┌──────────────┐
    │  产品视角     │  ╲    │   ╱  │  技术视角     │
    │  Product     │────── │ ─────│  Technical   │
    └──────┬───────┘ P↔T   │      └──────┬───────┘
           │        映射   │             │
           │               │             │
     ┌─────┴─────┐   ┌────┴────┐  ┌─────┴─────┐
     │ P↔D 映射  │   │ B↔D 映射│  │ T↔D 映射  │
     └─────┬─────┘   └────┬────┘  └─────┬─────┘
           │               │             │
           └───────────────┼─────────────┘
                    ┌──────┴───────┐
                    │  数据视角     │
                    │  Data        │
                    └──────────────┘
```
