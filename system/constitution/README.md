# system/constitution — 系统知识库宪法与治理层

本目录约束 **组织级 `system/`** 知识库中的术语边界、槽位约定与架构治理入口，与 [`architecture/`](../architecture/README.md)、[`application-APPNAME/`](../application-APPNAME/README.md) 并列。

**与中央库 `application/` 的职责边界**：四视角实体命名、IDEA-ID 等 **应用侧 SSOT** 仍以中央库 **`application/constitution/standards/`** 为准；本目录只描述 **系统树内** 如何组织镜像与槽位，不重复定义实体 ID 规则。人类可读的设计摘录见 **[`../../application/DESIGN.md`](../../application/DESIGN.md)**；命名细则见 **[`../../application/constitution/standards/naming-conventions.md`](../../application/constitution/standards/naming-conventions.md)**。
